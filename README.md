### Basiques
- [] Suivi des habitudes quotidiennes
- [] Stockage des habitudes dans un fichier JSON
- [] Ajouter une habitude
- [] Supprimer une habitude
- [] Marquer une habitude comme faite
- [] Afficher l'historique
- [] Enregistrer le nom de l'utilisateur à suivre (limiter à 1 utilisateur)
- [] Afficher une courbe de progression dans les habitudes faites sur un mois.
